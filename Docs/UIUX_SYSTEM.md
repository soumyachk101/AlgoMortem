# UI/UX System: "The Logic Lab"

## 1. Aesthetic Goal
The app should feel like a **scientific instrument** or a **high-end debugger**. It's not a playground; it's a lab for dissecting thoughts.

## 2. Core UI Concepts

### 2.1. The "Ghost Trace"
When a user makes a mistake, the AI doesn't just show text. It highlights the cell in the grid that is logically inconsistent with a "ghostly" red glow.

### 2.2. The "Breakthrough" Animation
When the logic is finally correct, the grid transitions into a code editor with a smooth "shatter" animation, symbolizing the mental block breaking.

### 2.3. The "Anti-Hint" Terminal
Hints appear as if they are being typed out in real-time in a console at the bottom-right.
